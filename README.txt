🚀 HOW TO RUN THE WEBSITE LOCALLY
────────────────────────────────────────────────────────────────
1. Extract all files to a folder on your computer
2. Open "index.html" with any modern web browser:
   - Double-click the file, OR
   - Right-click → Open with → Your Browser
3. Navigate through pages using the menu

For local development:
   - Use a local server like Live Server (VS Code extension)
   - Or use Python: python -m http.server 8000
   - Then open: http://localhost:8000

🔧 HOW TO CUSTOMIZE
────────────────────────────────────────────────────────────────

1. CHANGE SITE NAME & BRANDING
   ──────────────────────────────
   - Open each HTML file (index, about, contact, download)
   - Find <title> tag and update
   - Find "RBX FPS Unlocker" text and replace
   - Update footer with your name/brand

2. UPDATE LINKS
   ──────────────────────────────
   A) Official Website Link:
      - Search for: https://robloxfpsunlocker.org/
      - Replace with your official site URL

   B) Download Link:
      - Open: download.html
      - Find: https://robloxfpsunlocker.org/download/
      - Replace with your download URL

   C) Google Form Link:
      - Open: contact.html
      - Find: https://docs.google.com/forms/d/e/1FAIpQLSf...
      - Replace with your Google Form link

   D) GitHub Link:
      - Search all files for: https://github.com/isabellagreen968
      - Replace with your GitHub profile URL

3. MODIFY CONTENT & TEXT
   ──────────────────────────────
   - All text is in HTML files
   - Edit headings, paragraphs directly in the HTML
   - Keep semantic HTML structure (header, section, footer)
   - Maintain responsive class names

4. CHANGE COLORS
   ──────────────────────────────
   Open "style.css" and modify these variables:
   
   --color-primary: #EDF6FD;    → Light backgrounds
   --color-white: #FFFFFF;       → White elements
   --color-accent: #FF8080;      → Buttons, highlights
   --color-dark: #001A32;        → Text, navbar

5. ADJUST FONTS
   ──────────────────────────────
   Current font: Poppins (from Google Fonts)
   
   To change:
   - Open each HTML file
   - Find: fonts.googleapis.com/css2?family=Poppins
   - Replace "Poppins" with desired font name
   - Update in style.css: --font-primary

6. ADD/REMOVE SECTIONS
   ──────────────────────────────
   - Each section has class="section-name"
   - Copy entire <section> block to duplicate
   - Remove unwanted sections completely
   - Styles will auto-apply based on classes

🛠️ TROUBLESHOOTING
────────────────────────────────────────────────────────────────

PROBLEM: Menu not working on mobile
SOLUTION: Ensure script.js is loaded correctly

PROBLEM: Styles not applying
SOLUTION: Check that style.css is in the same folder

PROBLEM: Links not working
SOLUTION: Verify all HTML files are in the same directory

PROBLEM: Fonts not loading
SOLUTION: Requires internet connection for Google Fonts
